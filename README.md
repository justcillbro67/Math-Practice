# Math Practice

A simple math practice tool with two parts — a web app and a terminal program.

## Web App

Open `index.html` in your browser or visit the live site:

**https://justcillbro67.github.io/Math-Practice/**

Pick a number from the dropdown and hit Go — it shows the full multiplication table from 1 to 20.

## Terminal (Node.js)

Make sure you have [Node.js](https://nodejs.org) installed, then run:

```bash
node table.js 5
```

Replace `5` with any number you want.

### Example Output

```
Multiplication Table of 5:

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 20 = 100
```
